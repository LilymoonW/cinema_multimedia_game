## A Quick Godot Game For my Cinema Creative Project
Inspiration in class we were talking about parables and allegories. We had an exercise to create a parable or alegory for our class so we came up with a simple story that subverts expectations. I like playing on the idea of a user thinking they are the protagonist like the genre typically holds and subverting that to make the user think about what they are doing. Think of black mirror episodes and games like omori and undertale. I actually don't play many games myself but i watch alot of game streams!
So given that I wanted the art to be very cute and adorable.
I took alot of inspo from online and finally created this colorful space.

I have a very basic attack mechanic where pip the sparkle character which acts as the players companion cheers you on to kill and save the world
playing on the typical theme of games which is to kill and conquere for glory. However if you take time to observe you see that these blobs might not be so evil hinted by how easy they are to kill.
pip continues to enable you and cheers you on as you gather hope by killing more corrupted ones!
The boss fight is this evil figure but sometimes flashes of another appearce shows which should indicate that what you see may not be actually what is occuring. That these evil curropted ones might not be so corrupted.

There are a few endings to this short game but the mechanics are prretty simple just meant for a fun little project!
